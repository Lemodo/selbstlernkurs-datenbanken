Aggregatfunktionen - Zusammenfassung von Daten:

Aggregatfunktionen ermöglichen die Zusammenfassung von Daten in einer Tabelle. Hier sind einige gängige Aggregatfunktionen:

  • COUNT: Zählt die Anzahl der Datensätze.
  • SUM: Summiert Werte in einer Spalte.
  • AVG: Berechnet den Durchschnitt von Werten in einer Spalte.
  • MIN: Gibt den kleinsten Wert in einer Spalte zurück.
  • MAX: Gibt den größten Wert in einer Spalte zurück.

Beispiel COUNT:


              SELECT COUNT(Spalte) FROM Tabelle;

Beispiel AVG:


              SELECT AVG(Wert) FROM Tabelle;
