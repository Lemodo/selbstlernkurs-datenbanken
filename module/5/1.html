Modul 5: Abfragesprache SQL

Einführung in SQL:

SQL (Structured Query Language) ist eine speziell entwickelte Programmiersprache für das Management von relationellen Datenbanken. Mit SQL können Benutzer Daten abfragen, bearbeiten und verwalten.

SQL umfasst verschiedene Befehle, darunter SELECT (um Daten abzurufen), INSERT (um Daten einzufügen), UPDATE (um Daten zu aktualisieren) und DELETE (um Daten zu löschen).

Beispielsweise sieht ein SQL-Befehl um alle daten aus der Tabelle "data" herauszugeben so aus:

SELECT * FROM data;


Man jedoch auch Daten aus mehreren Tabellen abrufen:

SELECT Spalte1, Spalte2 FROM Tabelle WHERE Bedingung;


Hier wurde auch noch ein weiterer elementarer Bestandteil von SQL, die Bedingung genutzt.
Wie du Bedingungen und Operatoren benutzt lernst du im nächsten Abschnitt.
