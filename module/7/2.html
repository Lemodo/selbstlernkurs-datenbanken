Modul 7: Anwendung

Implementierung von ER-Diagrammen und Relationsschemata

  1. ER-Diagramm erstellen: Verstehen Sie, wie Entitäten, Beziehungen und Attribute in einem ER-Diagramm dargestellt werden. Nutzen Sie Tools wie draw.io oder Lucidchart für die Erstellung.
  2. Relationsschemata definieren: Lernen Sie, wie Sie aus einem ER-Diagramm Relationsschemata ableiten. Beachten Sie die Normalisierung, um die Datenbank effizient zu strukturieren.
  3. Umsetzung in SQL: Erfahren Sie, wie Sie die definierten Relationsschemata in SQL-Anweisungen umsetzen. Nutzen Sie CREATE TABLE, ALTER TABLE usw.
